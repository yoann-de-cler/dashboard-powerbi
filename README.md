# Créer un tableau de bord dynamique avec Power BI

## Objectif
Suivre l’avancement et la performance des projets à travers des KPI décisionnels (coût, durée, livrables) afin d’identifier les projets en retard, à risque ou conformes.

---

## Travail réalisé
- Analyse du besoin métier et définition des KPI de suivi de projet
- Nettoyage et transformation des données avec Power Query
- Modélisation de données relationnelles multi-niveaux (projet / phase)
- Création de mesures DAX pour le suivi des indicateurs clés
- Conception de dashboards interactifs sous Power BI
- Mise à jour du modèle de données et du dashboard à partir d’une nouvelle version des données sources

---

## Dashboard Power BI
Le dashboard est disponible sous 2 formats :
- Version de présentation => *Dashboard Power BI (PDF)*
- Dashboard interactif => *Dashboard Power BI (pbix)*

---

## Outils
- Power BI
- Power Query

---

## Résultats
- Recommandation stratégique
- Vu à l'échelle des différentes hiérarchie
- Filtrable sur tous les éléments des projets (date, type, pays, région, phase...)
